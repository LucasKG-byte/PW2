3 Formas de inserir CSS em um arquivo HTML

CSS Externo


Ele cria um arquivo separado com extensão .css e vincule-o ao HTML usando a tag link dentro do head

body {

background-color: #f0f0f0;
font-family: Arial, sans-serif;

}

h1 {

color: #333;

}

2. CSS Interno:


O CSS é escrito dentro da tag "style" no próprio HTML.

"DOCTYPE html"

"html lang="pt-BR""

"head"

"meta charset="UTF-8""

"meta name="viewport" content="width=device-width, initial-scale=1.0"

"title'Exemplo CSS Externo"/title"

"link rel="stylesheet" href="style.css">"

"/head"

"body"

"Olá, Mundo"

"/body"

"/html"

3. CSS Inline:


O estilo é aplicado diretamente no elemento usando o atributo style.

"h1 style="color: red; background-color: yellow;">Olá, Mundo!/h1"
